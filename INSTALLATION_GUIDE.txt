TROUBLESHOOTING
================================================================================

Problem: "Port 5000 already in use"
Solution: Edit app.py and change port to 5001, 5002, etc.

Problem: "ModuleNotFoundError: No module named 'flask'"
Solution: Run: pip install -r requirements.txt

Problem: "Database is locked"
Solution: Delete database/db.sqlite3 and restart the app

Problem: "Charts not rendering"
Solution: 
  1. Clear browser cache (Ctrl+Shift+Delete)
  2. Hard refresh page (Ctrl+F5)
  3. Check browser console (F12) for errors

Problem: "Can't connect to localhost:5000"
Solution:
  1. Check if Flask is still running (no error in terminal)
  2. Check firewall settings
  3. Try http://127.0.0.1:5000 instead

Problem: "Login not working"
Solution:
  1. Check database was initialized (check for db.sqlite3)
  2. Try demo/demo123 credentials
  3. Check browser console for errors
